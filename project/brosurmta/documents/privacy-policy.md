# Privacy Policy — Brosur MTA Lengkap

**Versi Aplikasi:** 3.0.0 ke atas  
**Terakhir Diperbarui:** 28 Maret 2026  
**Berlaku Efektif:** 28 Maret 2026

---

## 1. Pendahuluan

Selamat datang di **Brosur MTA Lengkap**. Kami berkomitmen untuk melindungi privasi Anda. Kebijakan privasi ini menjelaskan secara transparan bagaimana aplikasi kami bekerja dan data apa saja yang terlibat.

> **Penting:** Mulai versi 3.0.0, kami telah **menghapus seluruh iklan** dan **tidak menggunakan layanan analitik** apapun. Aplikasi ini dirancang untuk berfungsi sepenuhnya tanpa mengumpulkan data pribadi Anda.

---

## 2. Data yang Kami Kumpulkan

### ✅ Kami TIDAK Mengumpulkan:

- Data pribadi (nama, email, nomor telepon, dsb.)
- Data lokasi
- Data perangkat yang dapat mengidentifikasi pengguna
- Riwayat penggunaan atau statistik
- Data analitik apapun
- Informasi iklan atau ID pengiklan
- Cookie atau pelacak pihak ketiga

### 📋 Ringkasan:

**Aplikasi Brosur MTA Lengkap tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna dalam bentuk apapun.**

---

## 3. Izin yang Digunakan

Aplikasi ini memerlukan beberapa izin sistem Android untuk menjalankan fungsi utamanya. Berikut penjelasan lengkap setiap izin:

### 🌐 Izin Jaringan

| Izin                   | Tujuan                                                       |
| ---------------------- | ------------------------------------------------------------ |
| `INTERNET`             | Mengunduh konten brosur dari server MTA                      |
| `ACCESS_NETWORK_STATE` | Memeriksa apakah koneksi internet tersedia sebelum mengunduh |
| `ACCESS_WIFI_STATE`    | Mendeteksi status koneksi Wi-Fi untuk optimasi unduhan       |

> Koneksi internet digunakan **hanya** untuk mengambil konten brosur. Tidak ada data pengguna yang dikirim ke server manapun.

---

### 🎵 Izin Pemutaran Audio di Latar Belakang

| Izin                                | Tujuan                                                                             |
| ----------------------------------- | ---------------------------------------------------------------------------------- |
| `WAKE_LOCK`                         | Mencegah perangkat tidur saat audio sedang diputar                                 |
| `FOREGROUND_SERVICE`                | Menjalankan layanan pemutaran audio saat aplikasi di latar belakang                |
| `FOREGROUND_SERVICE_MEDIA_PLAYBACK` | Khusus Android 14+: izin eksplisit untuk layanan pemutaran media di latar belakang |

> Izin audio ini digunakan **semata-mata** agar pengguna dapat mendengarkan konten audio MTA secara terus-menerus, termasuk saat layar dikunci atau aplikasi diminimalkan. Tidak ada rekaman suara atau akses mikrofon.

---

## 4. Pihak Ketiga

Mulai versi **3.0.0**:

- ❌ **Tidak ada SDK iklan** (AdMob, Facebook Ads, dsb.)
- ❌ **Tidak ada SDK analitik** (Firebase Analytics, Mixpanel, dsb.)
- ❌ **Tidak ada SDK pelacakan** (Adjust, AppsFlyer, dsb.)
- ❌ **Tidak ada layanan crash reporting** pihak ketiga

Aplikasi ini beroperasi sepenuhnya mandiri tanpa keterlibatan layanan pihak ketiga yang mengumpulkan data.

---

## 5. Keamanan Data

Karena aplikasi ini **tidak mengumpulkan data apapun**, tidak ada risiko kebocoran atau penyalahgunaan data pengguna. Semua konten yang diunduh disimpan secara lokal di perangkat Anda dan sepenuhnya berada dalam kendali Anda.

---

## 6. Anak-Anak

Aplikasi ini aman digunakan oleh semua kalangan, termasuk anak-anak di bawah 13 tahun. Kami tidak mengumpulkan data dari siapapun, tanpa terkecuali.
