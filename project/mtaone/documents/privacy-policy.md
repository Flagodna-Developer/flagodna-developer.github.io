# Privacy Policy — MTA ONE

**Versi Aplikasi:** 3.0.0 ke atas  
**Terakhir Diperbarui:** 28 Maret 2026  
**Berlaku Efektif:** 28 Maret 2026  
**Developer:** PT. FlagoDNA Developer Labs

## 1. Pendahuluan

Selamat datang di **MTA ONE** — aplikasi all-in-one untuk mendukung kegiatan belajar dan ibadah bersama Majlis Tafsir Al-Quran (MTA). Kebijakan privasi ini menjelaskan secara transparan izin yang digunakan aplikasi, tujuannya, dan bagaimana kami mengelola data Anda.

---

## 2. Data yang Kami Kumpulkan

### 📍 Lokasi

Digunakan **hanya** untuk fitur **Arah Kiblat**. Data lokasi diproses langsung di perangkat Anda dan **tidak dikirim ke server manapun**.

| Izin                     | Tujuan                                                        |
| ------------------------ | ------------------------------------------------------------- |
| `ACCESS_FINE_LOCATION`   | Menentukan koordinat GPS akurat untuk perhitungan arah kiblat |
| `ACCESS_COARSE_LOCATION` | Alternatif lokasi perkiraan jika GPS tidak tersedia           |

> ⚠️ Lokasi Anda **tidak disimpan**, **tidak diunggah**, dan **tidak dibagikan** ke pihak manapun.

---

### 🌐 Jaringan Internet

| Izin                   | Tujuan                                                                |
| ---------------------- | --------------------------------------------------------------------- |
| `INTERNET`             | Streaming MTA TV, Radio Persada FM, brosur, berita, dan konten online |
| `ACCESS_NETWORK_STATE` | Memeriksa ketersediaan koneksi internet                               |
| `ACCESS_WIFI_STATE`    | Mendeteksi status Wi-Fi untuk optimasi koneksi                        |

---

### 🎵 Audio & Layanan Latar Belakang

| Izin                                | Tujuan                                                                       |
| ----------------------------------- | ---------------------------------------------------------------------------- |
| `WAKE_LOCK`                         | Mencegah perangkat tidur saat audio sedang diputar                           |
| `FOREGROUND_SERVICE`                | Menjalankan pemutaran audio (MTA TV / Radio) saat aplikasi di latar belakang |
| `FOREGROUND_SERVICE_MEDIA_PLAYBACK` | Android 14+: izin eksplisit untuk layanan pemutaran media                    |

---

### 🕌 Widget Jadwal Shalat

| Izin                                   | Tujuan                                                                                                           |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `RECEIVE_BOOT_COMPLETED`               | Memulihkan widget jadwal shalat secara otomatis setelah perangkat restart                                        |
| `REQUEST_IGNORE_BATTERY_OPTIMIZATIONS` | Memastikan widget jadwal shalat & notifikasi waktu shalat berjalan tepat waktu tanpa terganggu penghemat baterai |

---

### 📸 Kamera _(Hanya Versi di Bawah 3.0.0 — Sudah Dihapus)_

| Izin     | Tujuan                                            |
| -------- | ------------------------------------------------- |
| `CAMERA` | ~~Digunakan pada fitur Stories di versi < 3.0.0~~ |

> ✅ **Mulai versi 3.0.0**, fitur Stories telah **dihapus sepenuhnya**. Izin kamera **tidak lagi aktif** pada versi terbaru. Jika izin ini masih muncul, harap **hapus dan instal ulang** aplikasi untuk memperbarui ke versi terbaru.

---

### 💾 Penyimpanan _(Hanya Android 9 ke Bawah — Legacy)_

| Izin                     | Tujuan                                                              |
| ------------------------ | ------------------------------------------------------------------- |
| `WRITE_EXTERNAL_STORAGE` | Menyimpan file unduhan (hanya berlaku s.d. Android 9 / SDK 28)      |
| `READ_EXTERNAL_STORAGE`  | Membaca file yang tersimpan (hanya berlaku s.d. Android 9 / SDK 28) |

> Pada Android 10 ke atas, izin penyimpanan ini **tidak digunakan**.

## 4. Keamanan Data

Seluruh data sensitif (termasuk lokasi) diproses secara **lokal di perangkat** dan tidak dikirimkan ke server eksternal. Koneksi ke server MTA (untuk konten streaming dan brosur) menggunakan protokol terenkripsi.

---

## 5. Anak-Anak

Aplikasi ini dirating **3+** dan aman digunakan oleh semua kalangan. Kami tidak mengumpulkan data pribadi dari siapapun, termasuk anak-anak.
